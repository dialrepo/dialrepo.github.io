AccionesDescripción Enlace CU.Permisos
Exportar

Genera el informe asociado al listado de acuerdo al formato de exportación seleccionado

Si se selecciona el formato de exportación "csv" se generará el informe REP-ETI-0140 - Listado general de solicitudes de evaluación - Formato csv  pasando como parámetros los bloques de datos seleccionados (aquellos que tengan el check correspondiente marcado).

Si se selecciona el formato de exportación "xlsx" se generará el informe REP-ETI-0141 - Listado general de solicitudes de evaluación - Formato xlsx pasando como parámetros los bloques de datos seleccionados (aquellos que tengan el check correspondiente marcado).
