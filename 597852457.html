Exportar

Genera el informe asociado al listado de acuerdo al formato de exportación seleccionado

Si se selecciona el formato de exportación "csv" se generará el informe REP-CSP-0031 - Listado general de convocatorias - Formato csv  pasando como parámetros los bloques de datos seleccionados (aquellos que tengan el check correspondiente marcado).

Si se selecciona el formato de exportación "xlsx" se generará el informe REP-CSP-0030 - Listado general de convocatorias - Formato xlsx pasando como parámetros los bloques de datos seleccionados (aquellos que tengan el check correspondiente marcado).
