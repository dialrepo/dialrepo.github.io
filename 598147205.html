Hércules : Proceso 3 : Automatización de procesos de gestiones documentales parametrizables

Created by Juan Tardón Martínez, last modified by Usuario genérico para HERCULES on nov 15, 2022

Este proceso se divide en tres subprocesos, cada uno de ellos aparece detallado en las siguientes secciones:

  1. Extracción de bases reguladoras
  2. Extracción de convocatorias
  3. Extracción e identificación de concesiones

Sólo el subproceso relacionado con la extracción de las convocatorias insertará información en el SGI.

El diseño de la base de datos relacionado con este proceso se encuentra en la siguiente sección: Diseño de base de datos para proceso 3

Los requisitos generales del proceso se encuentran en la siguiente sección: Requisitos

  • Proceso de extracción e identificación de concesiones: https://github.com/hercules-rpa/orquestador/blob/main/model/process/ProcessExtractAward.py
