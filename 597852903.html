Hércules : SGI - Interface integración

Created by Usuario desconocido (alicia.alvarez), last modified by Usuario desconocido (lorena.alvarez) on abr 18, 2022

El SGI dispone de tres mecanismos de integración para aquellas universidades que lo quieran implantar:

  • Una serie de métodos expuestos a través de un API REST denominado ESB que se describen aquí: SGI - ESB
  • Unas consultas que se pueden utilizar para llevar a cabo acciones en los sistemas de la Universidad que lo implante y que se describen aquí: SGI - Consultas integración
  • Unas vistas para poder consultar cierta información del SGI y que se describen aquí: SGI - Vistas
