Definición y objetivos

El SGI requiere la disponibilidad del listado de países para mostrarlos en los módulos del SGI, generalmente en combos.

Descripción integración

Parámetros de entrada: la solicitud no requiere parámetros de entrada. 

Parámetros de salida: listado de países. Para cada elemento se devolverá:

  • Identificador
  • Nombre

Diseño técnico

En el siguiente enlace se muestra el diseño del modelo lógico y el diseño de la API del Sistema de Gestión de la estructura Orgánica SGI - ESB - SGO.

El servicio concreto del API que cubre este requisito es SGI - ESB - SGO - Países - Listar.
